MOCK CERTIFICATES NOTICE
================================================================================

Current state: MOCK MODE
  - Passes are NOT signed with Apple certificate
  - Signature file contains placeholder "UNSIGNED_MOCK_SIGNATURE"
  - Pass structure is valid and testable
  - Passes will NOT work on real Apple devices until signed

To enable real signing:
  1. Obtain Apple Pass Certificate from Apple Developer
  2. Export as signerCert.pem and signerKey.pem
  3. Place in certs/ directory
  4. WWDR Intermediate Certificate (wwdr.pem) optional
  5. Engine will auto-detect and use real certificates
